angular的过滤器

业务开发中经常会用到过滤器，现在在这里总结一下angular过滤器的用法，angular的过滤器是非常简单的，分为内置和自定义两种，下面分别介绍这两种过滤器的用法。

angular内置的过滤器

angular有9种内置的过滤器

  1. currency (货币处理)

    使用currency可以为数字加上货币符号，默认为美元符号，也可以传入所需的符号，例如传入人民币

    {{num | currency: '￥'}}

  2. date (时间格式化)

    时间格式化是我用的比较多的一种，{ {date | date : ‘yyyy-MM-dd hh:mm:ss EEEE’} }，y M d h m s E 分别表示 年 月 日 时 分 秒 星期，你可以随意组合他们。比如{ {date | date : ‘yyyy年MM月dd日 hh:mm:ss’} }，{ {date | date : ‘yyyy年MM月dd日’} }，{ {date | date : ‘hh:mm:ss’} }，只要有关键参数就可以随意组合。

  1. filter (匹配字符串)

    这个名叫filter的filter很容易让人搞混，这个filter可以用来处理一个数组，过滤出含有某个字符串的元素，返回一个过滤后的数组。处理的数组可以是字符串数组，可也以是对象数组。如果是对象数组，可以匹配属性的值。

  2. json（格式化json对象）

    json过滤器可以将js对象格式化为json字符串，我貌似从来没用过这个功能，我们从后台取回的数据一般都是json格式的，数据处理也都放在js里，官网上说它可以用来进行调试，好像也就只有这个作用了。

    <div ng-bind="item | json"></div>

  3. limitTo (限制数组长度或者字符串长度)

    limitTo用来截取数组或者字符串，接收一个参数用来指定截取的长度，如果参数是负值，则从数组尾部开始截取。这个过滤器只能从头或者尾开始截取，感觉有点鸡肋。

    <tr ng-repeat="item in items | limitTo: 3"></tr>//截取数组的前三个元素

  4. lowercase（小写）

    把数据转化为全部小写。同样是很鸡肋的一个filter，没有参数，只能把整个字符串变为小写。

    <div ng-bind="item | lowercase"></div>

  5. uppercase（大写）

    把数据转化为全部大写。同样是很鸡肋的一个filter，没有参数，只能把整个字符串变为大写。

    <div ng-bind="item | uppercase"></div>

  6. number（格式化数字）

    number过滤器可以为一个数字加上千位分隔，例如123,456,789。可接受一个参数，指定float类型保留几位小数。

    <span>{{num | number:2}}</span>//保留两位小数

  7. orderBy（排序）

    orderBy过滤器可以将一个数组中的元素按照某个字段属性重新排序，可以接受一个参数来指定排序规则。参数可以是一个字符串，表示以该属性名称进行排序。可以是一个函数，函数会遍历待排序的数组，并将返回的结果作为angular库函数中comparator的参数，进行比较排序。还可以是一个数组，表示依次按数组中的属性值进行排序（若按第一项比较的值相等，再按第二项比较）。还是拿上面的items举例：

    <tr ng-repeat="item in items | orderBy:'age'"></tr>//按照age的属性值进行排序，若是-age，则倒序。
<tr ng-repeat="item in items | orderBy: orderFunc"></tr>//按照函数的返回值进行排序
<tr ng-repeat="item in items | orderBy: ['age','name']"></tr>//按照age的属性值进行排序，如果age相同，按照name进行排序

内置的过滤器就是这些，angular内置的过滤器并不是万能的，实际上好多都基本用不上，我们日常开发中用的最多还是自定义的过滤器。

angular的自定义过滤器

自定义过滤器的结构

{{带过滤数据 | 过滤器名：参数1：参数2：参数3.....}}    

app.filter('过滤器名', function () {
    return function (待过滤数据, 参数1：参数2：参数3.....) {
        //过滤操作
        return  已过滤数据;
    }
});

下面举一个具体的栗子

reverse是过滤器的名字，后面跟的是过滤器的方法声明，子啊方法中返回一个方法：

{{ name | reverse:true }}

app.filter("reverse",function(){
    return function(input,uppercase){
      var out = "";
      for(var i=0 ; i<input.length; i++){
        out = input.charAt(i)+out;
      }
      if(uppercase){
        out = out.toUpperCase();
      }
      return out;
    }
});

内部返回的方法包含了两个参数，一个是输入的值，就是我们过滤器接受的值。input就是其中name的值
后面的参数是可选的，这里接受uppercase这个bool值，判断是否要大小写切换。

filter的使用方法

  1. 在模板中使用filter

    我们可以直接在{ { } }中使用filter，在表达式后面用 | 分割：

    {{ items | filter }}

    filter也可以多个连用，上一个filter的输出将作为下一个filter的输入：

    {{ items | filter1 | filter2 | ... }}

    filter可以接受参数，多个参数用:分割：

    {{ items | filter:argument1:argument2:... }}

    最常用的还是在指令中使用filter，先对数组进行过滤处理，然后在循环输出：

    <div ng-repeat="item in items | filter"></div>

  2. 在controller和servie中使用filter

    在controller中使用filter，首先要在controller中注入filter，例如上面的reverse：

    app.controller('test',function($scope,reverse){
  $scope.name = reverse('XiaoMing',true);
}

    当需要的filter比较多的时候，可以通过angular提供的$filter服务来调用所需的filter，只需要注入一个$filter就够了，使用方法如下：

    app.controller('testC',function($scope,$filter){
  $scope.name = $filter('reverse')('XiaoMing',true);
  $scope.date = $filter('date')(new Date(),"yyyy-MM-dd hh:mm:ss EEEE");
}

小结

以上就是关于angular中过滤器的一些基本用法，都非常的简单，但是不自己整理一遍还是挺容易忘的，整理的过程中会发现很多平时可能注意不到的细节。
